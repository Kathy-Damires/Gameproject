# Prompts para Generar Sprites con IA (Leonardo.ai o Copilot)

## Configuracion General
- Modelo: **Leonardo Phoenix** o **Leonardo Diffusion XL** (Leonardo.ai) / **DALL-E 3** (Copilot)
- Fondo: Activar **Transparent Background** siempre (o fondo blanco + remove.bg)
- Negative prompt: `text, words, letters, watermark, signature`
- Genera 4 variaciones, elegí la mejor
- **Estilo visual:** sci-fi dark theme, neon glow edges, polished digital art, mobile idle game UI
- **Referencia:** Adjuntar screenshot del juego en Copilot para mantener consistencia

2. Usa el MISMO nombre de archivo que el placeholder para reemplazar automaticamente
3. Mantene el mismo seed/estilo en toda la sesion
4. Las cartas de eras 4-9 seguí el mismo patron que las sesiones 18-20
5. Los panels/botones/barras de UI NO necesitan IA — los placeholders estan bien
